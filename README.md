# Mirror_DiscTPanzer

Сделан в рамках курса по неклассичесским логикам студентами направления ФИИТ: 
Погорелов Александр, Поповская Влада, Якубовский Илья, Бурмистров Артём, Руднев Александр
Ноябрь 2017 - декабрь 2017
